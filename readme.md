## React-typescript-starter
    Starter kit for React and Typescript projects. Bundled with Webpack.

### Project structure:
     .
    ├── assets
    ├── node_modules
    ├── public
    |       └── index.html
    ├── src
    |       ├── components
    |       ├── containers
    |       ├── App.tsx
    |       ├── index.tsx
    |       └── utils.ts
    ├── .gitignore
    ├── package-lock.json
    ├── package.json
    ├── readme.md
    ├── tsconfig.json
    └── webpack.config.ts

### Installation:
- Start development server: `npm run dev`
>Runs at port `9000`
- Generate deploy bundle: `npm run build`
